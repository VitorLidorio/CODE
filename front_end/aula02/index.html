(header) : topo do site

(h1>, (h2), ... (h6) : tags de título por ordem de relevância
(p> : parágrafo
(br /> : quebra de linha
(img) : imagem
(table>, (tr), (td) : tabela (dados tabulares)

(a> : hiper-ligação, link ou âncora
(input>, (select) ou (textarea) : campos de formulário
(button> : botões
(strong): negrito
